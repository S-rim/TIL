# html속성
## 이미지

attribute                               | 기술
--------------------------------------- | --------------------------------------
src                                     | 이미지 파일의 경로
alt                                     | 파일이 없을 때 표시되는 글
width                                   | 너비
height                                  | 높이

## 미디어

attribute                               | 기술
--------------------------------------- | --------------------------------------
src                                     | 미디어 파일의 경로
preload                                 | 재생 전 음악파일을 불러오는 지 지정
autoplay                                | 음악 파일을 자동재생할 지 지정 
loop                                    | 음악을 반복할 지 지정
loop                                    | 음악 재생 도구를 표시할 것인지 지정

## 비디오

attribute                               | 기술
--------------------------------------- | --------------------------------------
src                                     | 동영상 파일의 경로
poster                                  | 동영상을 준비할 때 보일 이미지의 경로
autoplay                                | 음악 파일을 자동재생할 지 지정 
loop                                    | 반복할 것인지 지정
controls                                | 재생 도구를 표시할 것인지 지정
preload                                 | 동영상 파일을 모두 불러올 것인지 지정
width                                   | 동영사 너비
height                                  | 동영상 높이
