# MVC

####Model -> Controller <- View

- 컨트롤러는 모델에 명령을 보내 모델의 상태를 변경할 수 있다. 

        ex) 워드 프로세서에서 문서 편집 

- 컨트롤러가 관련된 뷰에 명령을 보냄으로써 모델의 표시 방법을 바꿀 수 있다. 
        
        ex) 문서 스크롤

- 모델은 모델의 상태에 변화가 있을 때 컨트롤러와 뷰에 통보한다. 
            
            이를 통해서 뷰는 최신의 결과를 보여줄 수 있고
            컨트롤러는 모델의 변화에 따라
            추가·제거·수정할 수 있다. 
            
- 뷰는 사용자가 볼 결과물을 생성하기 위해 모델로부터 정보를 얻어 온다.
